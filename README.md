# Salon Management System

## Overview

A collaborative graduation project developed as part of the Information Systems Diploma program at King Khalid University.

The system helps manage salon operations including appointments, services, customers, and administration through a web-based platform.

---

## Technologies

- PHP
- HTML5
- CSS3
- JavaScript
- MySQL
- XAMPP
- phpMyAdmin

---

## Features

- User Authentication
- Appointment Booking
- Service Management
- Customer Management
- Admin Dashboard
